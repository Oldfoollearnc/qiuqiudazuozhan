# 球球大作战 🏀

一个基于 Canvas 的 Agar.io 风格网页游戏，单 HTML 文件，开箱即玩。

## 在线游玩

<https://oldfoollearnc.github.io/qiuqiudazuozhan/>

## 操作

| 操作 | 按键 |
|------|------|
| 移动 | 鼠标移动 |
| 分裂 | 空格（长按连续分裂） |
| 吐球 | W |
| 加速 | Shift / 右键 |
| 聊天 | Enter（1-7 发送快捷消息，Esc 关闭） |
| 退出 | Esc |

## 特性

- **游戏模式**：经典模式 / 大乱斗 / 无尽模式
- **AI 对手**：三种人格（激进/均衡/防守），数量可调（8/14/22）
- **皮肤系统**：9 种皮肤（纯色 / 凤凰 / 雷电 / 天使 / 恶魔 / 机甲 / 极光 / 星辰 / 幽靈），各有独特的粒子特效和动画
- **等级系统**：7 段位 × 50 级，解锁颜色奖励
- **成就系统**：9 项成就
- **排行榜**：实时显示前 10 名
- **小地图**：显示玩家、尖刺、超级食物位置
- **聊天系统**：7 条预设消息，数字键发送
- **回放系统**：记录最近 5 分钟对局
- **观战模式**：死亡后可观看第一名
- **自定义光标**：游戏内鼠标定位指示器
- **音效系统**：8 种合成音效

## 本地运行

直接双击 `index.html` 在浏览器中打开即可，无需任何服务器。

## 构建

纯前端项目，无构建工具，无依赖。
